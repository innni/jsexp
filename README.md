kickstart
=========

High Productivity Frontend Development Workflow

Prerequisites
--------------

- [Node.js](http://nodejs.org/)
- [Bower](http://bower.io/) 
- [Grunt](http://http://gruntjs.com/)

Installation
--------------

```sh
git clone https://github.com/innni/kickstart.git
cd kickstart
npm install
bower install
```

When you are ready...

```sh
grunt watch
```

and happy coding!
